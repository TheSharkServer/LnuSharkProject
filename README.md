# LnuSharkProject
采用Django框架搭建的网页服务，提供辽宁大学自动填报服务。

## 项目简介
疫情期间在校闲来无事，制作该项目仅用于学习Django框架及爬虫技术，提供DEMO供大家讨论，有相关问题和意见可以通过Issues反馈给我。

## 项目特色功能
 - 对接辽宁大学疫情通实现SharkSever账号与学号验证与绑定
 - 支持在线设置自动填报内容
 - 每日自动填报
 - 填报结果邮件通知
